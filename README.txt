2η Προγραμματιστική Εργασία - Υλοποίηση των αλγορίθμων συσταδοποίησης K-means / K-medoids για πολυγωνικές καμπύλες στη γλώσσα C/C++

Χρήστος Σκουλίκας-Ανδρούτσος - 1115201200164
Βίκτωρ Τσακαλάκης - 1115201200182

Το πρόγραμμα υλοποιήθηκε στη C, σύμφωνα με τις οδηγίες που δόθηκαν.
Συνεργαστήκαμε από κοντά για την ανάπτυξή του, αλλά χρησιμοποιήθηκε git ως version control.

Το αρχείο main.c περιέχει τις κεντρικές λειτουργίες του προγράμματος,
το metrics.c περιέχει τις μετρικές συναρτήσεις DTW, Frechet, ευκλίδειας απόστασης και το silhouette,
τα initialization.c, assignment.c και update.c τις αντίστοιχες λειτουργίες του αλγορίθμου K-means.
Τα αντίστοιχα header files περιέχουν δηλώσεις των σχετικών data structures και των συναρτήσεων.

Συμπεριλαμβάνεται makefile, οι εντολές είναι make για μεταγλώττιση, make clean για διαγραφή object files.

Το πρόγραμμα εξετάστηκε με valgrind για αποφυγή όλων των memory leaks.

Δοκιμάστηκε με dataset πλήθους 100 καμπυλών.
