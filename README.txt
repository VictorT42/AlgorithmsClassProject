Το πρόγραμμα υλοποιήθηκε στη C, σύμφωνα με τις οδηγίες που δόθηκαν.
Συνεργαστήκαμε από κοντά για την ανάπτυξή του, αλλά χρησιμοποιήθηκε git ως version control.

Το αρχείο main.c περιέχει τις κεντρικές λειτουργίες του προγράμματος,
το hash.c περιέχει τις συναρτήσεις της δομής hash table που δημιουργήθηκε για το πρόγραμμα,
το metrics.c περιέχει τις μετρικές συναρτήσεις DTW, Frechet και ευκλίδειας απόστασης,
το curves.c περιέχει τη συνάρτηση που μετατρέπει μία καμπύλη σε διάνυσμα grid curves, τη συνάρτηση που βρίσκει
την αληθινή κοντινότερη καμπύλη για ένα query και έναν απλό αλγόριθμο ταξινόμισης (insert sort) για
τη λεξικογραφική ταξινόμιση των R-nearest καμπυλών.
Τα αντίστοιχα header files περιέχουν δηλώσεις των σχετικών data structures και των συναρτήσεων.

Συμπεριλαμβάνεται makefile, οι εντολές είναι make για μεταγλώττιση, make clean για διαγραφή object files.

Το πρόγραμμα εξετάστηκε με valgrind για αποφυγή όλων των memory leaks.


Σημειώνεται πως επιστρέφονται στην main (αλλά δεν εμφανίζονται) οι αποστάσεις όλων των R-nearest
στα radius queries. Αν αποδειχθεί περιττό, θα αφαιρεθεί στο δεύτερο μέρος της εργασίας.
